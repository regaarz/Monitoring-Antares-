Proyek ini adalah sistem Internet of Things (IoT) yang berfungsi untuk memantau suhu dan kelembapan lingkungan menggunakan sensor DHT22 serta memungkinkan pengendalian lampu LED dari jarak jauh melalui platform IoT Antares. Data sensor dikirimkan (Publish) dari mikrokontroler ESP32 ke Antares, dan perintah kontrol LED diterima (Subscribe) dari Antares.

Aplikasi Android berfungsi sebagai antarmuka pengguna (User Interface) untuk menampilkan data sensor terbaru dan mengirimkan perintah ON/OFF ke LED.
